# 🎼 תִּזְמוֹרֶת הַתּוֹדוֹת — חֲוָויַת מִשְׁתַּמֵּשׁ

פּוֹסְטְיוֹן עִצּוּבִי שֶׁמַּרְחִיב אֶת הַ"מִּית" בְּכָל שׁוּרָה וּפוֹתֵחַ שַׁעֲרֵי הוֹדָיָה כְּתִזְמוֹרֶת רְבֵי־כְּלִים.

## 🧭 מַפַּת מַעֲבָרִים
- **אֲבִיב שֶׁל הוֹדָיָה → יְסוֹדוֹת הַטֶּבַע → הַלֵּב → הַפְּנִימִיּוֹת**
- **נֵס וְצְמִיחָה → חֶסֶד וְרִפּוּי → קְדוּשָּׁה → תְּקוּמָה → הַשִּׂיא**
- מַעֲבָרִים נַעֲשִׂים בְּגְרֵייְדְיֶינְט עָדִין וּפִידְבֶּק צְבָעִי מֵחָם לְקָר וּלְעֻמָּתוֹ.

## 🎨 אָרוֹן הַכְּלֵיִּים
- **בְּרִיחֵי צֵל:** מַדְרֵג קְטָרִים (4px, 12px, 24px) לַהֶעְמָקָה וְתָמִיד מוֹסִיף inset דַּקִּיק לְאוֹר פְּנִימִי.
- **טִיפּוֹגְרַפְיָה:** הַרְמוֹנְיָה בֵּין פָנִים סֶרִיפִיִּים (Frank Ruhl Libre / Playfair Display Hebrew) לְלְלֹא סֶרִיף (Assistant / Heebo) לְהַבְדָּלַת טֶקְסְט־עֲמֹק לְמִידְעִי.
- **אִיקוֹנוֹגְרַפְיָה:** אֵמוֹגִ'י מַנְגִּינִים כְּסִימָנֵי תִּוּוּחַ — צִפּוֹר לִישׁוּעָה, זְרִיחָה לַתְּחִלָּה, לַהֲבָה לַנֵּס.

## 📝 תַּבְנִית שׁוּרָה מֻבְנֵית
1. **אִיקוֹן מַנְגִּין** (🕊️ / 🌿 / ✨) — הַמַּקְלֵט הָרִשׁוֹן
2. **פֹּעַל בָּרוּר** בְּזְמַן הוֹוֶה: "אוֹמֵר", "מוֹדֶה", "מַפְנֶה", "מְשַׁחְרֵר"
3. **חִבּוּר תֵּמָתִי**: יֶסֶד הַשַּׁעַר (טֶבַע / לֵב / נֵס וכו')
4. **הֶד רַךְ**: תְּנוּעַת צֶל־אוֹר בַּמִּלִּים (לְדֻגְמָה: "בְּלַחַשׁ רָךְ")

## 🪄 מִקְצָב פִּידְבֶּק
- **אֲנִימַצְיָה רַכָּה:** בְּהַרְחָפָה צֶל חִיצוֹנִי וּבְהַקְלָדָה צֵל פְּנִימִי נוֹסָף (0 0 0 12px rgba(..., 0.1)).
- **קוֹל:** הוֹסָפַת סְטְרִינְגִּים קַלִּים (mp3 קָל) בְּרֶמֶז, מַתְאִים לְרִטְם הַשְּׁטֵף.
- **מִשׁוֹרַר הַשּׁוּרוֹת:** הַדְגָּשַׁת שׁוּרָה נוֹכְחִית עִם בָּר קָטָן בְּצֶבַע הַפָּלֶטָּה שֶׁל הַשַּׁעַר.

## 🔗 חִבּוּרִים לִנְאוֹגְרַפְיָּה קַיֶּמֶת
- **אִינְדֶּקְס תּוֹכֶן:** שִׁכְבַת נְוִיגַצְיָה בַּצַּד יָמִין שֶׁמְּצַיֶּנֶת אֵיזֶה שַׁעַר מִתְקָרֵב.
- **גִּרְסָה מְנוּקֶּדֶת:** שִׁמּוּר טַאיְפּוֹגְרַפְיָה בְּנִקּוּד בַּמִּסְמָךְ הַחוֹוָיָתִי לְאִמּוּת שְׁלֵמוּת הַלָּשׁוֹן.
- **מַדְפֶּסֶת:** יִצִּירַת פִּידְבֶּק לְשִׁכְבַת הַדְּיוֹ בְּגְּרֵייְדְיֶינְט כְּאִלּוּ הַצֵּל נִטְבָּל בַּצֶּבַע.

---

> **כָּל שׁוּרָה מְנַגֶּנֶת כַּמֵּיתָר — כַּאֲשֶׁר הַהוֹדָיָה מִתְרַחֶבֶת, הַמַּקְלֵט הוֹפֵךְ לְתִזְמוֹרֶת שֶׁל אוֹר.**
